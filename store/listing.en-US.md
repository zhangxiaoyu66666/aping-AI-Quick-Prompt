## Short Description

A Windows prompt workbench that turns rough intent, OCR text, templates, and Skills into model-ready prompts.

## Description

AI Quick Prompt is a Windows desktop prompt workbench for people who move between ChatGPT, Claude, Gemini, local OpenAI-compatible models, image models, video models, and coding agents.

It helps you turn rough requirements, OCR text, clipboard context, local templates, and mounted `SKILL.md` workflows into clearer prompts that are ready to send to a model.

Use it for:

- ChatGPT, Claude, Gemini, local model, and OpenAI-compatible model prompts.
- Text-to-image, video prompt, AI coding, writing, rewriting, and summarization workflows.
- Local OCR from screenshots or images, then using the recognized text as editable prompt context.
- Favorite prompts and reusable templates.
- Mounted Skill workflows that guide a model through a specific task process.

Highlights:

- Chat-style requirement refinement.
- Chinese and English prompt output panes.
- Optimization targets for general LLM, image prompts, Jimeng, Veo 3, AI coding, Skill workflows, and custom platforms.
- Local Fire Eye OCR, with Windows Media OCR as fallback only.
- OpenAI-compatible provider configuration.
- Favorite prompts, local templates, Skill mounting, and JSON import/export.
- Global hotkey, system tray, and compact prompt window.
- API keys stored in Windows Credential Manager.

Privacy:

AI Quick Prompt is local-first by default. OCR runs locally unless you explicitly send image context to a configured model provider. Text or image context is sent only when you send a model request and model egress is enabled. Please configure your provider according to its own terms and privacy requirements.
