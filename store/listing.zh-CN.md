# Microsoft Store Listing - zh-CN

## 产品名称

啊拼

## 简短描述

AI 快速提示词工作台，把需求、OCR、模板和 Skill 整理成可直接发送给模型的提示词。

## 完整描述

啊拼是一款 Windows 桌面 AI 快速提示词工作台。它帮助你把一句粗略需求、OCR 文本、剪贴板上下文、本地模板和挂载的 Skill 工作流整理成更清晰、更可执行的提示词。

适合这些场景：

- 写给 ChatGPT、Claude、Gemini、本地模型或 OpenAI 兼容模型的任务提示词。
- 文生图、视频提示词、AI 编程、写作、润色和总结。
- 从截图或图片中本地识别文字，再把 OCR 内容作为提示词上下文。
- 收藏常用提示词，快速复用模板。
- 挂载 `SKILL.md` 工作流，让模型按指定流程完成任务。

主要功能：

- 聊天式需求优化。
- 中文与英文提示词并排输出。
- 优化目标：通用 LLM、文生图、即梦、Veo 3、AI 编程、Skill 体系和自定义平台。
- 本地 Fire Eye OCR，Windows Media OCR 仅作为兜底方案。
- OpenAI 兼容模型提供商配置。
- 常用提示词收藏、本地模板、Skill 挂载和 JSON 导入导出。
- 全局快捷键、系统托盘和紧凑窗口。
- API Key 存储在 Windows Credential Manager。

隐私说明：

啊拼默认本地优先。OCR 默认在本机完成。只有当你明确发送请求并启用模型外发时，文本或图片上下文才会发送到你配置的模型提供商。请根据自己的模型服务条款和隐私要求配置提供商。
