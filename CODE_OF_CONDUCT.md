# Code Of Conduct

This project follows a simple contributor standard:

- Be respectful and specific.
- Critique code and behavior, not people.
- Avoid harassment, threats, personal attacks, and discriminatory language.
- Keep bug reports and reviews actionable.
- Respect privacy; do not post another user's prompts, screenshots, API keys, or local files.

Maintainers may remove comments, issues, or pull requests that violate these expectations.
