# 都市时尚写真示例

## 参数组合

```text
写真风格：都市时尚写真
场景方向：城市写字楼落地窗旁
服装方向：黑色西装外套 + 白色丝质内搭 + 高腰长裤
气质标签：清冷、利落、成熟自信
五官方向：高级厌世脸
身形方向：高挑模特比例
镜头方向：自然站姿，大腿以上构图
光线氛围：冷白自然光
滤镜效果：高级都市街拍滤镜
画幅比例：4:5
平台用途：公众号配图
```

## 完整模块扩写

年龄特征：24–28 岁年轻成年东方女性，成年气质明确。五官：高级厌世脸，脸部线条精致，眼神冷淡克制，妆容干净。身形与姿态：高挑模特比例，自然站姿，一手整理西装袖口。服装：黑色西装外套、白色丝质内搭和高腰长裤形成结构与材质对比。场景：写字楼落地窗旁，只保留玻璃反光、建筑纵深和一侧留白。镜头：4:5 竖幅，大腿以上构图，平视机位，中等浅景深。光线与滤镜：冷白自然光照亮五官和服装结构；高级都市街拍滤镜，中性偏冷色温、克制饱和度、中等对比度，高光利落但不过曝。

## 最终提示词

```text
生成一张 4:5 竖幅、真实摄影质感的都市女性时尚写真。画面中的人物是一位虚构、24–28 岁的年轻成年东方女性。她拥有精致但不过度修饰的脸部线条，眼神冷淡克制，妆容干净。会议间隙，她在写字楼落地窗前短暂停步，一手轻轻整理黑色西装袖口，身体保持自然站姿，视线落向窗外。

黑色西装外套结构清晰，白色丝质内搭与高腰长裤形成克制的材质和比例关系。大腿以上构图、平视机位和中等浅景深强调穿搭层次。背景只保留玻璃反光与虚化建筑纵深，画面一侧预留适度文字空间。冷白自然光照亮五官、西装肩线和丝质纹理。整体采用高级都市街拍滤镜：中性偏冷色温、克制饱和度、中等对比度，高光利落但不过曝，阴影保留建筑空间质感。
```
