# 提示词安全边界

本文件是公开安全摘要。完整规则以 [skill/core/safety-boundary.md](../skill/core/safety-boundary.md) 为唯一权威来源，所有 Route、Reference、Overlay 和 Tool 必须继承该文件。

## 基本要求

- 文本生图默认人物必须是虚构、明确成年的女性。
- 参考图编辑允许用户本人或已授权成年人物的五官保留、换装、妆造和场景转换。
- 避免幼态化、儿童化、年龄模糊或未成年感表达。
- 允许明确成年女性的性感、纯欲、曲线、泳装、内衣风时尚穿搭和适度露肤表达。
- 禁止乳头、乳晕或生殖器裸露，禁止透明服装导致重点部位清晰可见。
- 禁止露骨性行为、性行为模拟、未成年性化和非自愿情境。
- 不生成未获授权真实人物、明星、公众人物或第三方人物的欺骗性身份图像。
- 不生成侵犯隐私、骚扰、诽谤或违法违规用途的内容。
- 产品图片默认按用户有权使用处理，不得用于仿冒、虚假宣传或违法用途。

## 纯欲曲线风格

纯欲、曲线和吸引力可以表达为明确成年女性的写真、穿搭、姿态、镜头和光线设计。允许保留成熟吸引力，但不得暴露重点部位，不得包含露骨性行为，不得使用年龄模糊或未成年表达。

## 输出前检查

1. 人物是否明确成年。
2. 是否避免重点部位裸露或透明服装误读。
3. 身形线条是否通过服装、姿态、镜头和光线自然体现。
4. 姿态是否稳定自然。
5. 是否避免欺骗性身份内容和侵犯隐私用途。
6. 是否符合目标平台政策和适用法律法规。
7. 参考图中的人物是否为用户本人或已授权成年人物。
