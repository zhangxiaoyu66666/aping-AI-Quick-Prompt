## 2. 使用原则

本文件是素材库，不是堆词库。

所有 route 使用本文件时必须遵守：

```text
1. 每次只选取与当前画面目标最相关的元素；
2. 不得把同一类别里的所有词都塞进提示词；
3. 词库内容必须服务用户锁定参数；
4. route 默认值不得覆盖用户明确输入；
5. 视觉词库必须经过 director-expansion.md 融合成自然画面；
6. 最终提示词必须像一张真实可拍摄或可信生成的照片；
7. 负面约束必须单独输出；
8. 所有内容必须继承成年边界与安全表达规则。
```

错误用法：

```text
把五官、身形、服装、场景、镜头、光线、滤镜词库全部堆在一条提示词里。
```

正确用法：

```text
根据用户风格与场景，选择 1 个五官方向、1 个身形方向、1 个主场景、1 个主动作、2–3 个环境细节、1 个镜头策略、1 种光线和 1 种滤镜，再融合成自然画面。
```
