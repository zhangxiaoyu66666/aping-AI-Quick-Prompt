## 1. 风格定位

电商服装模特图 / 试衣镜用于根据用户上传或描述的服装，生成模特穿着该服装的稳定展示图。

本 route 的核心不是生成一张普通好看的女性写真，而是：

```text
服装还原优先；
商品展示优先；
颜色、版型、材质、图案、结构和装饰稳定优先；
模特、姿势、场景、光线都必须服务服装展示。
```

视觉关键词：

```text
电商服装展示
AI 试衣镜
上传服装还原
模特穿着效果
服装完整清晰
不色差
版型准确
材质准确
商品主图
小红书种草
干净背景
摄影棚
服装上身图
穿搭展示
真实摄影质感
```

画面目标：

```text
根据用户上传服装或描述服装生成模特穿着效果；
服装颜色、品类、版型、材质、图案、装饰必须尽量保留；
模特只是服装展示载体；
人物成年气质明确；
场景干净，不抢服装；
姿势稳定，不遮挡服装；
镜头能看清服装主体；
光线真实均匀，不造成严重色差；
适合电商主图、小红书种草、详情页、穿搭展示。
```

## 4. 路由边界

### 4.1 与普通女性写真区别

普通写真可以优先考虑人物美感、氛围和风格。

本 route 必须优先考虑：

```text
服装是否准确；
服装是否完整；
服装是否清楚；
服装是否被遮挡；
服装颜色是否接近原图；
服装版型是否被改变；
服装材质是否被误读；
是否适合商品展示。
```

如果用户只是写“生成一张都市时尚写真”，调用：

```text
skill/routes/fashion/urban-fashion.md
```

如果用户写“上传服装 / 电商主图 / 服装完整清晰 / 不要色差”，必须调用本 route。

---

### 4.2 与都市时尚写真区别

`都市时尚写真` 更偏：

```text
城市街拍；
穿搭氛围；
人物气质；
生活方式；
时尚感。
```

`电商服装模特图 / 试衣镜` 更偏：

```text
服装还原；
商品展示；
服装结构清晰；
颜色准确；
背景干净；
姿势稳定。
```

如果用户写：

```text
上传服装，女模，都市时尚，小红书种草，不要色差
```

处理：

```text
主 route：ecommerce-tryon
都市时尚：作为模特气质或场景参考
服装还原：最高优先
```

---

### 4.3 与影楼精修写真区别

影楼精修写真重点是人物精致度和棚拍完成度。

电商试衣镜重点是：

```text
商品服装上身效果；
颜色、材质、版型、结构；
服装清楚可售卖；
模特不能抢服装。
```

如果用户写“社交头像 / 高级精修 / 背景布”，调用影楼精修。
如果用户写“上传服装 / 商品展示 / 服装完整”，调用本 route。

---

### 4.4 与小红书种草图区别

小红书种草只是平台用途，不是主 route。

如果用户写：

```text
平台用途：小红书种草
```

本 route 的处理方式是：

```text
构图更生活化；
画面更自然；
背景更有分享感；
但服装还原仍然是第一优先级。
```

不得因为“小红书种草”而：

```text
改变服装颜色；
增加遮挡服装的外套；
使用复杂背景；
让人物姿势遮挡衣服；
把商品图变成普通生活写真。
```

---

### 4.5 与纯欲曲线生活照区别

如果用户上传的是贴身、吊带、短裤、内搭、泳装或贴身服装，系统仍必须按电商商品展示处理。

不得自动转入纯欲曲线 route。

正确处理：

```text
主 route：ecommerce-tryon
纯欲 / 清纯 / 轻熟：只作为模特气质
服装展示：最高优先
安全表达：服装完整、商品展示、非低俗
```

## 5. 优先级规则

本 route 的优先级顺序固定如下：

```text
P0：安全边界与成年边界
P1：用户上传服装 / 用户描述服装
P2：服装品类
P3：服装颜色
P4：服装版型
P5：服装材质
P6：服装图案 / 纹理 / 装饰
P7：服装完整度
P8：展示范围
P9：模特类型与气质
P10：姿势
P11：场景
P12：平台用途
P13：画面美感
```

说明：

```text
服装准确性永远高于模特漂亮；
服装完整度永远高于姿势好看；
颜色准确性永远高于滤镜氛围；
商品展示永远高于生活感；
用户上传服装永远高于系统默认服装。
```

## 7. 上传服装识别规则

如果用户上传服装图片，系统必须优先从上传服装中识别以下信息：

```text
服装品类：
上衣、外套、衬衫、T恤、针织、连衣裙、半裙、裤装、短裤、内搭、套装、运动装、汉服、旗袍、泳装等。

颜色：
主色、辅色、边缘色、印花色、纽扣 / 拉链 / 装饰色。

版型：
宽松、修身、短款、长款、高腰、低腰、直筒、A 字、包身、吊带、无袖、短袖、长袖、开衫、连体等。

材质：
棉、针织、牛仔、丝质、雪纺、皮革、蕾丝、网纱、缎面、亚麻、运动面料等。

结构：
领口、袖口、肩带、腰线、下摆、门襟、拉链、纽扣、口袋、开衩、抽绳、拼接、褶皱等。

装饰：
蕾丝、刺绣、印花、图案、蝴蝶结、金属扣、珍珠、亮片、压褶、花边等。
```

识别后必须在提示词中强调：

```text
必须穿着用户上传图片中的同款服装；
尽量保留原图服装的核心视觉特征；
颜色接近原图；
版型接近原图；
材质纹理接近原图；
装饰细节尽量保留；
服装完整清晰展示。
```

## 8. 参数锁定规则

用户明确填写的参数必须优先保留。

### 8.1 服装锁定

用户写：

```text
上传服装图片：已上传
补充要求：服装要完整清晰，不要有色差
```

必须输出：

```text
模特必须穿着用户上传图片中的同款服装，并尽量保留原图服装的颜色、品类、版型、材质、图案、纹理和装饰细节。服装主体必须完整清晰展示，避免色差，避免改变款式。
```

不得输出：

```text
类似风格服装；
同色系服装；
不同版型服装；
新增外套遮挡；
改变颜色以适配场景；
忽略上传服装细节。
```

---

### 8.2 模特类型锁定

用户写：

```text
模特类型：女模
```

必须保留女模。

允许扩写：

```text
画面主体是一位年轻成年东方女性模特，成年气质明确，外形自然干净，作为服装展示载体出现。
```

不得改为：

```text
男模；
儿童模特；
多人模特；
欧美模特；
虚拟假人；
只展示服装不出现人物。
```

除非用户明确要求“不出现人物”。

---

```text
模特气质：清纯
```

```text
模特五官清秀自然，妆容清透，整体气质清纯干净，但不幼态化，人物不抢服装主体。
```

```text

古风仙气；
```

## 10. 场景规则

本 route 场景必须服务服装展示。

### 10.1 电商主图默认场景

推荐：

```text
浅灰纯背景；
米白摄影棚；
干净白色背景；
低饱和极简室内；
浅色地面 + 纯色墙面。
```

扩写：

```text
背景采用简洁浅色摄影棚或纯色墙面，光线均匀柔和，无复杂道具和杂乱环境，服装主体清晰突出。
```

---

### 10.2 小红书种草默认场景

推荐：

```text
浅色室内；
极简卧室角落；
咖啡店外浅色墙；
干净生活化空间；
自然光窗边；
简洁街边墙面。
```

扩写：

```text
场景可以比电商主图更生活化，但必须保持背景干净、低干扰，不能遮挡服装主体，服装颜色和版型仍需准确。
```

---

### 10.3 详情页辅助场景

推荐：

```text
摄影棚正面；

背面展示；
局部材质特写；
不同角度展示；
浅色背景统一系列。
```

注意：

```text
如果用户只要求一张图，优先输出正面或轻侧身展示；
如果用户要求多张，可以拆分角度。
```

## 11. 姿势规则

电商试衣姿势必须简单稳定。

推荐姿势：

```text
正面自然站姿；
轻微侧身站姿；
双手自然垂落；
一只手轻扶衣摆；
一只手轻放腰侧但不遮挡服装；
自然站立看向镜头；
半身站姿；
上半身正面展示；
全身直立展示。
```

小红书种草可使用：

```text
自然靠墙；
轻微侧身；
手轻扶头发但不遮挡服装；
自然行走中停下；
轻扶包带但不能遮挡服装主体。
```

禁止姿势：

```text
双臂交叉遮挡胸前；
手臂大面积挡住上衣；
抱包遮挡服装；
低头导致头发遮挡领口；
复杂坐姿导致服装褶皱变形；
大幅扭腰导致版型失真；
夸张动作导致服装看不清；
低俗姿势；
身体部位特写。

```

## 24. 常见失败原因与修复

### 24.1 服装色差明显

原因：

```text
滤镜色偏过强；
光线过暖或过冷；
背景反光影响服装颜色；
没有明确锁定原图颜色。
```

修复：

```text
加入“服装颜色必须尽量接近用户上传原图，使用低色差真实电商摄影光，不使用强滤镜，不让环境光改变服装颜色。”
```

---

### 24.2 服装结构变了

原因：

```text
服装描述不够具体；
没有锁定领口、袖口、衣长、腰线、裤长或裙长；
姿势导致服装被遮挡。
```

修复：

```text
明确“保持原图服装版型，不改变领口高度、袖长、衣长、腰线、裤长、裙长和整体廓形。”
```

---

### 24.3 背景太复杂

原因：

```text
场景描述过度生活化；
道具太多；
平台用途被误解成生活写真。
```

修复：

```text
加入“背景简洁低干扰，避免复杂道具、杂乱环境、路人、招牌和强装饰元素。”
```

---

### 24.4 模特遮挡服装

原因：

```text
姿势过复杂；
手臂、包袋、头发遮挡；
动作链过多。
```

修复：

```text
使用正面自然站姿或轻侧身，双手自然垂落或轻扶衣摆，避免遮挡服装主体。
```

---

### 24.5 变成普通写真

原因：

```text
人物气质和氛围写得过强；
服装还原要求不够靠前；
场景过于写真化。
```

修复：

```text
在提示词前半段强调“模特只是服装展示载体，服装颜色、版型、材质、图案和装饰必须准确，服装主体完整清晰。”
```

## 25. 最终原则

电商服装模特图 / 试衣镜 route 的最终原则是：

```text
不是普通女性写真，而是服装商品展示；
不是模特越漂亮越好，而是服装越准确越好；
不是场景越丰富越好，而是背景越不干扰越好；
不是滤镜越强越好，而是颜色越准确越好；
不是姿势越复杂越好，而是服装越清楚越好；
不是类似款服装，而是尽量还原用户上传服装；
不是氛围优先，而是服装品类、颜色、版型、材质和装饰优先。
```

所有调用本 route 的任务必须遵守本文件。
