## 1. 风格定位

本 route 的核心不是简单堆叠“红金、华丽、宫廷、珠宝、女主”，而是建立一张具有高级古偶女主质感的华丽东方幻想角色写真：人物明艳但不艳俗，服装华丽但结构清楚，配色鲜明但不杂乱，光线明亮但不过曝，背景盛大但不抢主体。


## 4. 路由边界

### 4.1 与清冷仙气古风增强版区别

`清冷仙气古风增强版` 更偏：

```text
冷白；
疏离；
克制；
低饱和；
清冷距离感；
冷调柔光；
小面积点缀。
```

`明媚华贵古风增强版` 更偏：

```text
明艳；
华贵；
盛大；
红金或暖金；
女主高光；
宫廷华丽；
珠宝刺绣；
明亮柔光；
视觉冲击。
```

如果用户输入：

```text
清冷、冷白、疏离、不要喜庆、不要暖热
```

优先调用：

```text
skill/routes/fantasy/cold-xianxia-enhanced.md
```

如果用户输入：

```text
明媚、华贵、红金、盛唐、宫廷、女主感
```

调用本 route。

---

### 4.2 与基础古风仙侠美人图区别

`基础古风仙侠美人图` 是通用路线，可覆盖清雅、梦幻、柔美、华丽等方向。

本 route 必须明显强化：

```text
明艳东方五官；
华贵妆造；
更强女主感；
更高服装华丽度；
更明显珠宝、刺绣、头饰高光；
更适合角色海报和传播封面。
```

如果用户只写“古风仙侠美人图”，可调用基础 route。
如果用户写“明媚华贵 / 红金 / 盛唐 / 古偶女主 / 宫廷感”，调用本 route。

---

### 4.3 与东方丰腴写真区别

`东方丰腴写真` 重点是：

```text
成熟柔润；
丰腴协调；
东方古典身体美；
服装垂坠和含蓄曲线。
```

`明媚华贵古风增强版` 重点是：

```text
华丽古风角色完成度；
明艳女主气场；
红金或华丽配色；
宫廷 / 盛唐 / 古偶海报感。
```

如果用户写“明媚华贵 + 东方丰腴”，处理规则：

```text
若主风格明确为明媚华贵古风，则调用本 route；
东方丰腴只作为身形方向补充，通过腰封、衣料垂坠和大半身构图表达成熟柔润，不得低俗化。
```

---

### 4.4 与影楼精修写真区别

本 route 可以有精修完成度，但不是普通影楼古装。

必须避免：

```text
廉价影楼背景；
塑料古装；
艳俗头饰；
过度磨皮；
红金婚庆感；
现代礼服误入；

```

本 route 应强调：

```text
东方幻想角色写真；
唐风幻想古装结构；
高完成度妆造；
华丽但统一的配色；
明亮柔光；
古风空间虚化；
古偶女主海报质感。
```

---

### 4.5 与品牌 KV / 商业广告人像区别

本 route 主要用于古风角色人像。

如果用户输入：

```text

```
