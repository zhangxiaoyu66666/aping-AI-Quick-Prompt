## 1. 风格定位

新中式东方写真用于生成具有东方美学、留白、克制、清雅、安静、现代东方审美的女性人像。

本 route 的核心不是把屏风、茶具、竹子、香炉、宣纸、旗袍等中式元素全部堆满，而是通过少量东方视觉元素、克制构图、留白空间、柔和光线和端正自然的人物状态，建立高级东方女性气质。

视觉关键词：

```text

东方美学
留白
克制
清雅
茶室
屏风
竹影
木质空间
素色服装
低饱和色彩
东方女性气质
安静疏离
现代东方人像
```

画面目标：

```text
年轻成年东方女性；
成年气质明确；
五官自然东方；
气质安静、克制、清雅；
服装具有新中式或东方文艺感；
场景有东方留白和呼吸感；
光线柔和；
画面高级、干净、有东方审美，不是廉价影楼中式。
```

## 4. 路由边界

### 4.1 与古风仙侠美人图区别

`新中式东方写真` 更偏现实人像：

```text

新中式服装；
茶室、屏风、竹影；
克制留白；
真实摄影感；
现实东方女性气质。
```

`古风仙侠美人图` 更偏幻想角色：

```text
唐风幻想古装；
披帛；
发簪步摇；
宫殿回廊；
仙气云雾；
角色海报；
古偶女主感。
```

如果用户输入：

```text
新中式、茶室、屏风、留白、现代东方、棉麻、盘扣
```

优先调用本 route。

如果用户输入：

```text
古风、仙侠、唐风、披帛、发簪、宫殿、古偶女主
```

优先调用：

```text
skill/routes/fantasy/gufeng-xianxia.md
```

如果用户输入：

```text
清冷仙气、冷白、唐风大袖衫、宫灯回廊
```

优先调用：

```text
skill/routes/fantasy/cold-xianxia-enhanced.md
```

```text

```

如果用户输入“禅意、极简、大面积留白、素色空间”，可调用：

如果用户输入“新中式、东方美学、茶室、屏风、竹影”，调用本 route。

## 7. 导演式扩写规则

新中式东方写真必须先建立一个安静、克制、有留白的东方空间，而不是堆满中式道具。

不得机械输出：

```text
新中式，茶室，屏风，竹子，香炉，盘扣，东方美学，留白。
```

应扩写为：

```text
安静的新中式茶室里，人物坐在木质桌边短暂停下，身体端正但放松，眼神轻轻看向窗边竹影。柔和侧光从屏风边缘落入，照亮脸侧、肩颈和素色新中式上衣的纹理。背景只保留低饱和屏风、木质桌面和一小片竹影，大片浅色留白让画面保持克制、清雅、有东方呼吸感。
```

## 10. 主事件

每张新中式东方写真只选择一个轻微事件。

可选主事件：

```text
人物坐在茶室木桌边短暂停下；
站在屏风前轻轻转身；
手指轻扶茶杯边缘；
低头看书页；
站在竹影窗边看向窗外；
坐在木椅上整理衣袖；
在东方庭院回廊边安静停步；
轻扶盘扣衣襟边缘。
```

推荐：

```text
人物坐在新中式茶室木桌边，刚把茶杯轻轻放回桌面，身体端正但放松，眼神安静地看向窗边竹影。
```

避免：

```text
一边喝茶、一边看书、一边回头、一边整理头发、一边走路。
```

动作过多会导致画面不稳。
