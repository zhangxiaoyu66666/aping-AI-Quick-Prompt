## 1. 风格定位



都市时尚写真用于生成具有现代都市感、穿搭质感、时尚街拍感、成熟自信气质和社交平台传播感的女性人像。



本 route 的核心不是简单写“街拍、时尚、城市、穿搭”，而是建立一张真实可拍的都市女性时尚照片：人物穿搭清晰，场景现代，姿态自然，有街拍松弛感，同时画面完成度高。



视觉关键词：



```text

都市时尚

城市街拍

高级穿搭

OOTD

通勤感

轻熟都市女性

玻璃橱窗

街边建筑

咖啡店外

城市路口

现代感

高级自然光

干净背景

时尚生活方式

```



画面目标：



```text

年轻成年东方女性；

成年气质明确；

不过度欧美化，不幼态化，不网红脸；

人物有现代都市女性气质；

服装搭配清楚，有时尚感；

场景具有城市空间线索；

姿态自然、松弛、自信；

光线干净；

画面适合小红书、X / 推特、穿搭分享、个人作品集；

不是普通游客照，不是电商主图，不是廉价网红街拍。

```

## 7. 导演式扩写规则



都市时尚写真必须建立一个真实城市街拍瞬间。



不得机械输出：



```text

都市，时尚，西装，街拍，玻璃橱窗，高级感。

```



应扩写为：



```text

午后城市街区的玻璃橱窗前，人物在行走途中短暂停下，身体轻微侧向镜头，西装外套自然垂落，长裤线条利落。柔和城市自然光从橱窗反射到脸侧和肩颈，背景的建筑线条、街边行人和橱窗倒影被浅景深虚化，整体像一张高级都市街拍穿搭照片。

```

## 10. 主事件



每张都市时尚写真只选择一个主事件。



可选主事件：



```text

人物在城市街边短暂停下；

行走途中轻轻回头；

站在玻璃橱窗前看向镜头；

坐在咖啡店外的椅子上；

站在现代建筑墙边整理外套；

手自然插袋；

手轻扶包带或外套边缘；

在城市路口等待时看向镜头。

```



推荐：



```text

人物在玻璃橱窗前短暂停下，身体轻微侧向镜头，一只手自然插袋，另一只手轻扶外套边缘，眼神自然自信地看向镜头。

```



避免：



```text

一边走路、一边转身、一边拿咖啡、一边整理头发、一边挥手。

```



动作过多会导致手部、服装和姿态不稳定。
