## 1. 风格定位



影楼精修写真用于生成画面干净、光线稳定、人物精致、妆发完整、修图感较强但不廉价的女性人像。



本 route 的核心不是简单写“影楼、精修、棚拍、高级”，而是要建立一个稳定可控的棚拍或写真馆画面：人物是主体，光线干净，服装完整，妆发精致，背景简洁，整体有精修完成度，但避免塑料皮肤、过度磨皮、廉价背景和假面感。



视觉关键词：



```text

写真馆风格

棚拍柔光

高级人像

精致妆发

干净背景

人物主体清晰

皮肤干净但有质感

商业精修

高级女性写真

```



画面目标：



```text

年轻成年东方女性；

成年气质明确；

妆发完整精致；

五官清晰耐看；

服装结构清楚；

光线稳定柔和；

背景干净不抢主体；

精修但不塑料；

高级但不廉价；

适合头像、作品集、商业写真或社交平台展示。

```

## 7. 导演式扩写规则



影楼精修写真必须建立一个稳定、可拍摄的棚拍瞬间。



不得机械输出：



```text

影楼精修，棚拍，柔光，精致妆容，高级写真，背景布。

```



应扩写为：



```text

简洁摄影棚中，人物站在低饱和灰色背景布前，身体轻微侧向镜头，肩颈舒展，一只手自然扶在腰侧，另一只手轻轻放在裙摆边缘。柔和棚拍主光从侧前方落下，补光均匀覆盖脸部和衣料纹理，背景保持干净虚化，整体精致、高级、稳定，有写真馆精修完成度但不过度磨皮。

```

## 10. 主事件



每张影楼精修写真只选择一个轻微事件或稳定姿态。



可选主事件：



```text

人物站在背景布前自然停住；

轻轻整理袖口；

手扶裙摆边缘；

坐在简洁椅子上看向镜头；

拍摄间隙自然侧身；

轻扶耳饰或发丝；

站在柔光中看向镜头；

微微转头形成肩颈线条。

```



推荐：



```text

人物站在简洁背景前，身体轻微侧向镜头，肩颈舒展，一只手轻扶袖口，另一只手自然垂落，眼神平静地看向镜头。

```



避免：



```text

一边走路、一边转身、一边抬手、一边整理头发、一边看向远处。

```



动作过多会降低影楼精修稳定性。

头部轻轻转向镜头；

眼神稳定；

稳定；

自然；

不私房化。
