## 4. 路由边界

### 4.1 与健康运动线条写真区别

`活力运动写真` 更偏整体风格：

```text
健康；
阳光；
元气；
运动穿搭；
运动场景；
运动生活感；
社交平台传播感。
```

`健康运动线条写真` 更偏身形线条 route：

```text
肩背线条；
腰腹线条；
腿部线条；
运动身体比例；
健康曲线控制。
```

如果用户主要写：

```text
运动、活力、元气、户外、跑道、网球场、运动穿搭
```

调用本 route。

如果用户强烈强调：

```text
腹部线条、腿部线条、肩背、健身身材、运动曲线
```

可根据强度调用：

```text
skill/routes/curve/fitness-line.md
```

如果用户明确写“活力运动写真 + 腹部和腿部线条”，则保留本 route，并安全叠加健康线条表达。

---

### 4.2 与纯欲曲线生活照区别

活力运动写真可以有运动线条，但不能走纯欲路线。

本 route 应保持：

```text
健康；
阳光；
清爽；
自然；
运动状态；
```

如果用户明确输入：

```text
纯欲、冷白、锁骨、腰线、小腹、大腿、贴身吊带、身形吸引力
```

优先判断是否调用：

```text
skill/routes/curve/pure-desire-curve.md
```

但如果用户明确写“活力运动写真”，则保留运动主风格，将身形表达转译为健康运动线条。

---

## 10. 主事件

每张活力运动写真只选择一个主事件。

可选主事件：

```text
运动后短暂停在场边；
自然整理运动外套拉链；
站在跑道边看向镜头；
手扶运动水杯但不遮挡服装；
轻微拉伸肩颈；
网球场上刚停下脚步；
坐在场边短暂休息；
慢跑后停在阳光下。
```

推荐：

```text
人物在运动后短暂停在场边，身体自然站立，肩颈舒展，一只手轻扶运动外套拉链，另一只手自然垂落，眼神明朗地看向镜头。
```

避免：

```text
跳跃、奔跑、挥拍、转身、喝水、看镜头同时发生。
```

复杂动作容易导致肢体畸形、手部错误和服装变形。
