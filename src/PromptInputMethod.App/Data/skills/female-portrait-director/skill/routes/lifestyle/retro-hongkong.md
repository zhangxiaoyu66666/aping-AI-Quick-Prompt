## 1. 风格定位

本 route 的核心不是简单添加“红唇、卷发、霓虹、港风”几个词，而是要建立一张像港片电影剧照一样成立的画面。

```text

妆容复古但不廉价；
场景有年代感但不杂乱；

整体像旧港片中的一帧女主剧照。
```

## 4. 路由边界

### 4.1 与胶片复古写真区别

`复古港风写真` 更偏：

```text
港片女主；
明艳五官；
红唇或复古妆容；
旧香港街道或室内；
霓虹、茶餐厅、老式旅馆；
更强人物存在感。
```

`胶片复古写真` 更偏：

```text
旧照片感；
文艺复古；
暖棕低饱和；
老房间、旧旅馆、旧街道；
人物气质可以更安静、更文艺。
```

如果用户明确输入“港风、港片、红唇、旧香港、明艳浓颜”，优先调用本 route。

如果用户只输入“胶片、旧照片、暖棕、复古旅馆、低饱和”，优先调用：

```text
skill/routes/lifestyle/film-retro.md
```

---

### 4.2 与夜色霓虹写真区别

`复古港风写真` 的霓虹是旧港片氛围，不是赛博夜景。

本 route 可以有霓虹，但应保持：

```text
胶片质感；
人物明艳；
旧街电影感；
低饱和暖调；
年代感。
```

不得变成：

```text
赛博朋克；
冷艳夜店；
未来城市霓虹；
廉价夜拍；
过度蓝紫高饱和。
```

如果用户明确输入“赛博、未来感、城市夜景、强霓虹、雨夜街头、天桥”，可考虑：

```text
skill/routes/fashion/neon-night.md
```

但如果用户明确写“港风”，仍保留本 route。

---

### 4.3 与明艳女主 Overlay 区别

本 route 是主风格。

`明艳女主增强` 是 overlay，只能叠加气质，不替代本 route。

例如：

```text
复古港风写真 + 明艳女主增强
```

处理方式：

```text
仍然是复古港风写真；
只增强五官存在感、眼神、妆容和女主气场。
```

```text

头部轻轻转向镜头；
```

```text

不挑逗；
不低俗；
不刻意撩衣；
不做身体部位凝视；
不使用过度扭腰姿势。
```
