## 1. 风格定位

清纯生活照用于生成自然、干净、温柔、亲近、生活化、电影剧照感的女性人像。

本 route 的核心不是简单写“清纯、温柔、自然光”，而是建立一张像真实生活电影中被自然捕捉到的片段：人物不刻意摆拍，环境真实，光线柔和，表情自然，气质干净耐看。

视觉关键词：

```text
清纯生活照
电影生活剧照感
自然光
咖啡馆
窗边
奶油暖白
温柔
初恋感
邻家感
真实生活感
干净自然
轻微故事感
日常拍摄感
非影楼
非网红精修
```

画面目标：

```text
年轻成年东方女性；
成年气质明确；
不过度欧美化，不幼态化，不网红脸；
气质清纯、温柔、自然、亲近、耐看；
场景真实生活化；
服装干净日常；
姿态自然稳定；
光线柔和；
画面像生活电影中的一帧；
不是普通自拍，不是廉价影楼，不是强商业写真，不是 CG 插画。
```

## 7. 导演式扩写规则

清纯生活照必须建立一个真实生活中的轻微瞬间。

不得机械输出：

```text
清纯，咖啡馆，白色针织，温柔，自然光，奶油暖白。
```

应扩写为：

```text
午后咖啡馆靠窗座位，人物自然坐在浅色木桌旁，身体微微转向镜头，手轻轻放在咖啡杯旁边，肩膀放松，眼神温柔地看向镜头。窗边柔和自然光照亮她的脸侧、针织开衫纹理和浅色桌面，背景的玻璃窗与咖啡馆空间被浅景深虚化，整体像生活电影中被安静捕捉到的一帧。
```

## 10. 主事件

每张清纯生活照只选择一个主事件。

可选主事件：

```text
人物自然坐在咖啡馆靠窗座位；
手轻轻放在咖啡杯旁；
坐在窗边看向镜头；
低头看书页；
靠在桌边短暂停留；
坐在床边或椅子上看向窗外；
轻轻整理针织开衫袖口；
自然回头看向镜头。
```

推荐：

```text
人物自然坐在咖啡馆靠窗座位，身体轻微转向镜头，一只手轻放在桌面，另一只手自然搭在腿侧，眼神温柔地看向镜头。
```

避免：

```text
一边喝咖啡、一边看书、一边回头、一边整理头发、一边站起来。
```

动作过多会导致画面不稳定。

## 14. 服装扩写规则

清纯生活照服装必须干净、日常、柔和、自然。
不得过度暴露，不得情趣化，不得强商业化。

```text
内搭必须完整；
```
