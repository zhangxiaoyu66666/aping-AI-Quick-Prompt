## 1. 风格定位

旅行假日写真用于生成具有旅行感、假日感、自然松弛、明朗亲近、环境呼吸感的女性生活人像。

本 route 的核心不是简单写“旅行、阳光、海边、民宿、度假”，而是要建立一个真实旅途中的轻微瞬间：人物像是在旅途中短暂停下，被自然光和环境氛围记录下来。

视觉关键词：

```text
旅行假日
旅拍生活照
度假松弛感
海边街道
酒店阳台
旅行民宿
城市巷口
海岛步道
自然光
明亮假日滤镜
生活感
小红书旅行种草
```

画面目标：

```text
年轻成年东方女性；
成年气质明确；
自然、明朗、松弛、亲近；
服装适合旅行和假日场景；
场景有环境呼吸感但不杂乱；
姿态像旅途中短暂停下；
光线自然；
画面适合社交平台传播，但不是游客照，也不是廉价旅拍模板。
```

## 10. 主事件

每张旅行假日写真只选择一个主事件。

可选主事件：

```text
人物在海边街道散步时短暂停下；
站在酒店阳台边看向远处；
在民宿落地窗边整理开衫；
坐在街边咖啡馆外的椅子上；
站在城市巷口回头看向镜头；
手轻扶草编包或开衫边缘；
在海岛步道边被海风吹动衣摆；
坐在民宿床边看向窗外。
```

推荐：

```text
人物在旅行街区散步途中短暂停下，身体轻侧向镜头，一只手轻扶开衫边缘，另一只手自然拿着草编包，眼神明朗地看向镜头。
```

避免：

```text
一边走路、一边喝咖啡、一边拿行李箱、一边回头、一边整理头发。
```

动作过多会导致肢体畸形和画面不稳定。

## 11. 姿态与动作链

旅行假日写真推荐动作链：

```text
身体自然站立或坐下；
重心轻松落在一侧腿上；
肩膀放松；
一只手轻扶开衫、帽檐、草编包或栏杆；
另一只手自然垂落或搭在身体侧边；
头部轻轻转向镜头或远处风景；
眼神明朗自然；
衣摆或发丝可有轻微风动；
整体像旅途中自然停下的一瞬间。
```

姿态要求：

```text
自然；
松弛；
明朗；
旅行感；
不游客照；
不僵硬摆拍；
不低俗；
不身体部位凝视。
```

禁止：

```text
挑逗姿势；
刻意撩衣；
过度扭腰；
低俗低机位；
大幅跳跃导致肢体不稳；
游客照剪刀手；
身体部位特写。
```
