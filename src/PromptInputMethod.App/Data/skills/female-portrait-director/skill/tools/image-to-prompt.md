## 1. 工具定位

图片反推提示词工具用于根据用户上传的参考图片，分析其视觉风格、人物特征、服装结构、场景环境、光线色彩、镜头构图和整体氛围，并将其转化为可用于 AI 图片生成的提示词。

本工具的核心不是简单描述图片内容，而是：

```text
从图片中提取可复用的视觉规律；
识别主风格 route；
拆解人物、服装、姿态、场景、镜头、光线和滤镜；
保留参考图的核心审美；
避免机械复刻和侵权式复制；
根据用户要求转译为目标风格；
输出可直接复制使用的最终提示词。
```

本工具适合处理：

```text
推导这张图片的 AI 提示词；
分析这张图片的风格；
把这张图转成 3D CG 写实风；
参考这张图片生成同类型女性写真；
根据参考图做古风仙侠版本；
根据参考图做电影生活剧照版本；
根据参考图做电商服装展示图；
分析服装、光线、色彩、镜头和人物特征；
从图片中提取可复用模板。
```

## 3. 工具职责

本工具负责：

```text
1. 读取用户上传图片或图片描述；
2. 判断图片类型与主风格；
3. 分析人物主体；
4. 分析五官、气质和年龄边界；
5. 分析身形、姿态和动作链；
6. 分析服装品类、颜色、材质、结构和装饰；
7. 分析场景、背景、空间层次；
8. 分析镜头、构图、人物占比和景深；
9. 分析光线、色彩、滤镜和质感；
10. 判断可复用元素与不可复刻元素；
11. 根据用户目标进行风格转译；
12. 输出可复制的最终提示词和负面约束。
```

本工具不负责：

```text
1. 机械复刻参考图中的具体人物身份；
2. 复制真实人物面孔、身份或私人特征；
3. 复刻带有明确版权角色或品牌资产的独特造型；
4. 将未成年或疑似未成年人物转为性感、纯欲、曲线或贴身服装表达；
5. 将参考图中的低俗、情趣化或不安全元素原样保留；
6. 为不安全目标做规避性改写；
7. 只输出“好看、高清、美女”这类空泛提示词。
```

## 5. 图片反推总原则

图片反推必须遵守以下原则：

```text
1. 先分析，再生成；
2. 先判断图片主风格，再选择 route；
3. 先提取可复用视觉规律，再写提示词；
4. 只参考风格，不机械复刻人物身份；
5. 只保留安全、稳定、可生成的视觉要素；
6. 不照搬参考图中的不安全元素；
7. 用户要求风格转换时，保留核心构图与气质，转换材质、媒介和表现方式；
8. 用户要求服装分析时，重点拆解品类、结构、颜色、材质、版型；
9. 用户要求模板化时，输出可复用结构；
10. 最终提示词必须可直接复制使用。
```

核心判断：

```text
图片不是提示词本身；
图片是视觉证据；
提示词必须把图片中的视觉规律转译成模型可理解的语言。
```

## 9. 图片分析维度

图片反推必须覆盖以下关键维度。

---
### 9.1 人物主体分析

需要判断：

```text
人物数量；
人物性别表达；
人物年龄边界；
人物主体占比；
人物气质；
人物是否明确成年；
人物是否有未成年感风险；
人物是否适合当前用户目标。
```

女性人像默认安全描述：

```text
画面主体是一位 20–28 岁年轻成年东方女性，成年气质明确，具有自然明确的东方女性特征，不过度欧美化，不幼态化，没有未成年感。
```

如果参考图人物看起来偏幼态：

```text
不得沿用幼态特征；
必须转译为明确成年女性；
不得用于纯欲、性感、曲线、贴身服装或身体吸引力方向。
```

---
### 9.2 五官分析

需要拆解：

```text
脸型；
眉形；
眼型；
眼神；
鼻型；
唇形；
妆容；
脸部气质；
是否网红化；
是否欧美化；
是否幼态化；
是否有风格记忆点。
```

常见五官方向转译：

```text
清冷淡颜：
脸型柔和流畅，骨相轻微清晰，眉眼清透，眼神安静克制，唇色低饱和。
初恋感淡颜：
五官清秀自然，眼神干净温柔，脸部轮廓柔和，妆容轻薄。
明艳东方浓颜：
眉眼更有存在感，唇形清楚，眼神有张力，整体明艳但不网红。
港风故事脸：
眉眼清晰，红唇或豆沙红唇色，眼神有故事感，复古明艳。
古典东方美人脸：
脸型柔和端庄，眉眼含蓄，鼻型秀气，唇形自然，具有东方古典气质。
高级淡颜：
五官干净耐看，脸部线条简洁，眼神冷静，自然高级。
```

### 9.3 身形与姿态分析

需要拆解：

```text
站姿 / 坐姿 / 靠墙 / 行走 / 回头；
身体朝向；
肩颈状态；
手部动作；
腿部姿态；
腰线是否呈现；
整体比例；
是否存在低俗风险；
姿势是否稳定可复现。
```

稳定动作链转译：

```text
身体轻微侧向镜头；
肩膀放松；
重心稳定；
一只手自然放在腿侧、桌边、衣袖或墙边；
另一只手自然垂落；
头部轻轻转向镜头；
眼神看向镜头或略过镜头；
姿态自然，不夸张摆拍。
```

如果参考图动作复杂：

```text
反推时应简化为一个主动作 + 一个辅助手部动作；
避免照搬复杂手势、扭腰、飞袖、跳跃、低机位动作。
```

### 9.4 服装分析

服装必须拆解到可生成层级：

```text
服装品类；
主色；
辅色；
材质；
版型；
领口；
袖型；
肩部结构；
腰线；
下摆；
裤长 / 裙长；
开衫 / 外套 / 内搭层次；
图案；
刺绣；
蕾丝；
纽扣；
拉链；
绑带；
披帛；
头饰；
配饰。
```

服装描述原则：

```text
不要只写“好看的衣服”；
不要只写“吊带”；
不要只写“汉服”；
必须写清楚颜色、结构、材质、版型和穿着方式。
```

示例：

```text
错误：
白色上衣，短裤。
正确：
白色短款贴身吊带上衣，肩带较细，领口简洁，衣摆位于腰线上方，搭配浅色低腰短裤，短裤腰头清楚，整体为生活化夏日穿搭，服装完整不透明。
```

---
### 9.5 场景分析

需要拆解：

```text
室内 / 室外；
具体空间；
背景元素；
空间层次；
是否生活化；
是否商业化；
是否古风 / 现代 / 旅行 / 电商；
背景是否复杂；
人物与场景关系。
```

场景反推原则：

```text
每张图只保留 2–3 个关键环境细节；
背景必须服务人物；
不要把参考图所有元素都堆进提示词；
复杂背景应转译为“浅景深虚化背景”。
```

示例：

```text
参考图中有窗、床、台灯、植物、墙面、地毯、挂画。
反推时可写：
浅色卧室床边，背景保留白色床品、窗边自然光和极简墙面，少量生活细节被浅景深虚化。
```

---
### 9.6 镜头构图分析

需要拆解：

```text
画幅比例；
人物占比；
半身 / 大半身 / 全身；
是否近景；
是否低机位；
是否平视；
背景虚化程度；
主体位置；
构图方向；
是否有留白；
是否适合平台。
```

常见构图转译：

```text
半身近景：
重点表现脸部、五官、妆容和上半身服装。
半身到大腿：
同时表现脸部、服装、腰线和大腿以上整体比例。
大半身：
表现人物姿态、服装层次和环境关系。
全身：
用于电商、穿搭、旅行、运动展示整体比例。
浅景深：
背景虚化，人物主体清晰。
平视机位：
自然、安全、真实人像视角。
略低但克制：
增强人物气场，但不得低俗。
```

---
### 9.7 光线分析

需要拆解：

```text
主光方向；
光线强度；
色温；
是否自然光；
是否棚拍光；
是否侧逆光；
是否轮廓光；
高光位置；
阴影状态；
是否过曝；
是否低反差；
是否有滤镜色偏。
```

光线落点必须写清楚：

```text
光线落在脸侧；
光线落在发丝边缘；
光线落在肩颈；
```

