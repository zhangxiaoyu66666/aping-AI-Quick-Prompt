## 1. 工具定位

提示词优化工具用于处理用户已经提供的一条提示词、一个参数组合、一个半成品模板或一段失败提示词，并在不改变用户核心参数的前提下，将其优化为更稳定、更自然、更有导演感、更适合 AI 图片生成的最终提示词。

本工具的核心不是“重新创作一条新提示词”，而是：

```text
保留用户原意；
锁定用户参数；
识别原提示词主 route；
修复机械填参；
补足画面逻辑；
增强导演式扩写；
减少风格冲突；
提升出图稳定性；
补齐负面约束。
```

```text
优化这条提示词

这条提示词太机械了
不要改变原参数
保持原风格优化
帮我重写一下
让它更容易出图
提升稳定性

保留参数但扩写得更自然
帮我润色成可生成图片的提示词
这条提示词不够有导演感
```

典型用户输入：

```text
这条提示词太机械了，帮我优化一下，不要改变原参数。
帮我改得更稳定，服装、场景、五官都不要动。
我觉得这条提示词没有我们之前一起创作的效果，帮我按导演式扩写重写。
基于下面这条提示词，优化成更适合 AI 出图的最终版本。
```

## 3. 工具职责

本工具负责：

```text
1. 读取用户原始提示词；
2. 提取原始参数；
3. 判断任务类型和主 route；
4. 锁定用户明确参数；
5. 找出机械填参、风格冲突、缺少画面逻辑、缺少负面约束等问题；
6. 在不改变原始参数的基础上扩写；
7. 将关键词堆叠改成自然画面描述；
8. 补充人物、场景、服装、姿态、光线、镜头、滤镜之间的关系；
9. 补齐成年边界和安全边界；
10. 输出优化后的最终提示词和负面约束。
```

本工具不负责：

```text
1. 擅自替换用户风格；
2. 擅自改变服装、场景、五官、身形、镜头、画幅；
3. 把生活照改成影楼；
4. 把新中式改成古风仙侠；
5. 把电商服装图改成普通写真；
6. 把清纯改成纯欲；
7. 把纯欲优化成低俗性感；
8. 为了“更好看”删除用户已锁定内容；
9. 只写摘要，不给最终可复制提示词。
```

## 5. 优化总原则

提示词优化必须遵守以下原则：

```text
1. 先保护，再优化；
2. 先识别主 route，再扩写；
3. 先判断用户原意，再补全细节；
4. 只增强，不替换；
5. 只修复，不重造；
6. 缺失可补，明确不可改；
7. 风格要更清楚，参数不能被覆盖；
8. 提示词要像一张完整画面，不像字段列表；
9. 负面约束必须补齐；
10. 输出必须可复制使用。
```

## 6. 优化流程

每次执行提示词优化时，按以下流程处理：

```text
第一步：读取原始提示词
第二步：提取明确参数
第三步：判断主 route
第四步：锁定不可改参数
第五步：识别问题类型
第六步：修复结构问题
第七步：进行导演式扩写
第八步：补充缺失稳定项
第九步：补充安全边界
第十步：输出优化结果
```

```text
写真风格

配色方向
华丽度

```

```text
写真风格：清纯生活照
场景方向：咖啡馆窗边
服装方向：白色针织开衫
气质标签：清纯、温柔、自然
光线氛围：自然光
画幅比例：9:16
```

## 8. 参数锁定规则

用户原提示词中的明确参数必须锁定。

### 8.1 风格锁定

用户原提示词写：

```text
清冷仙气古风
```

不得优化成：

```text
明媚华贵古风
复古港风
都市时尚
新中式东方写真
```

只能在清冷仙气古风范围内扩写。

---
### 8.2 场景锁定

用户原提示词写：

```text
冷调宫灯回廊
```

不得优化成：

```text
云雾山水
现代咖啡馆
海边步道
法式公寓
摄影棚
```

只能扩写为：

```text
冷调宫灯回廊中，廊柱、宫灯、栏杆和远处帷幔被浅景深虚化，整体保持冷白基调。
```

---
### 8.3 服装锁定

用户原提示词写：

```text
华丽唐风大袖衫 + 披帛
```

不得优化成：

```text
旗袍
现代礼服
白衬衫
吊带连衣裙
影楼古装
```

只能扩写服装结构、材质、刺绣、披帛垂坠和层次。

---
### 8.4 五官锁定

用户原提示词写：

```text
清冷淡颜，丹凤眼，眼神安静
```

不得优化成：

```text
明艳浓颜
甜美元气脸
欧美脸
网红脸
幼态娃娃脸
```

只能扩写为：

```text
脸型柔和流畅，眉眼清透，眼型偏丹凤眼，眼尾微微上扬，眼神安静克制。
```

---
### 8.5 身形锁定

用户原提示词写：

```text
胸腰腿线条感，身形吸引力强度中
```

不得删掉，也不得低俗化。

正确优化：

```text
通过完整服装、轻侧身姿态、冷白自然光和大腿以上构图呈现胸腰腿整体比例和自然线条，不做身体局部特写。
```

---
### 8.6 画幅锁定

用户写：

```text
9:16
```

不得改成：

```text
1:1
4:5
16:9
2.35:1
```

纯欲、锁骨、腰线、小腹、大腿、冷白 → pure-desire-curve

古风、仙侠、唐风、披帛、花钿 → gufeng-xianxia

复古港风、茶餐厅、霓虹、红唇 → retro-hongkong
法式、慵懒、窗边公寓 → french-lazy

东方丰腴、旗袍、丰润、端庄 → oriental-voluptuous

## 10. 常见问题识别

优化前必须判断原提示词存在什么问题。

### 10.1 机械填参

表现：

```text
只是把字段拼起来；
没有主事件；
没有姿态动作链；
没有光线落点；
没有场景层次；
没有人物与环境关系；
没有镜头逻辑。
```

修复：

```text
加入时间切片、主事件、动作链、场景层次、光线落点、镜头构图。
```

---
### 10.2 风格不清

表现：

```text
清纯 + 纯欲 + 港风 + 新中式 + 古风同时出现；
没有主风格；
多个风格互相抢。
```

修复：

```text
确定一个主 route；
其他风格降级为气质、配色、滤镜或局部细节；
删除明显冲突项，或安全转译。
```

---
### 10.3 服装不稳定

表现：

```text
服装描述太短；
材质、颜色、版型、领口、袖口、下摆不清；
容易被模型替换；
电商服装没有锁定颜色和结构。
```

修复：

```text
补充颜色、材质、版型、结构、层次、穿着方式和禁止替换要求。
```

---
### 10.4 五官趋同

表现：

```text
只写美女；
只写精致五官；
所有风格都是同一张脸；
没有五官方向。
```

修复：

```text
补充明确五官方向，例如清冷淡颜、初恋感淡颜、明艳东方浓颜、古典东方美人脸、港风故事脸、高级淡颜等。
```

---
### 10.5 身形表达跑偏

表现：

```text
只写性感；
只写曲线；
没有成年边界；
没有服装完整；
容易低俗化或身体局部特写。
```

修复：

```text
转译为成年女性自然吸引力；
通过服装版型、姿态、镜头和光线表达；
禁止局部特写、低俗低机位和情趣化。
```

---
### 10.6 光线滤镜冲突

表现：

```text
冷白 + 暖黄；
清冷 + 红金喜庆；
电商 + 强胶片色偏；
自然光 + 夜景霓虹；
高曝光 + 五官清晰度不足。
```

修复：

```text
根据主 route 确定主光线；
冲突光线降级为局部点缀；
电商任务优先低色差真实光；
清冷任务降低暖色比例。
```

---
### 10.7 镜头不稳定

表现：

```text
同时要求全身、半身、近景、特写；
动作太多；
姿势复杂；
构图不明确。
```

修复：

```text
只保留一个主镜头；
动作链控制在 1–2 个动作；
优先稳定姿态；
避免身体局部特写和低俗低机位。
```

---
### 10.8 缺少负面约束

表现：

```text
没有限制幼态；
没有限制网红脸；
没有限制服装变形；
没有限制手部畸形；
没有限制低俗化；
没有 route 专属负面约束。
```

修复：

```text
补齐基础负面约束 + route 专属负面约束 + 用户问题相关负面约束。
```

## 11. 导演式优化规则

优化后的提示词必须从“字段拼接”变成“自然画面”。

### 11.1 必须补充时间切片

示例：

```text
午后咖啡馆靠窗自然光
夜色刚起的旧香港街角
冷调宫灯回廊中宫灯刚亮
阴天低反差落地窗边
海边傍晚柔和自然光
```

### 11.2 必须补充主事件

示例：

```text
人物坐在咖啡馆靠窗座位短暂停下
人物站在霓虹街角轻微回头
人物在宫殿回廊中轻扶披帛
人物靠在浅色墙边自然看向镜头
人物在城市街边行走中短暂停下
```

### 11.3 必须补充姿态动作链

动作链要简单：

```text
身体轻微侧向镜头；
肩膀放松；
一只手自然放在桌边 / 腿侧 / 衣袖边缘；
眼神看向镜头；
表情自然克制。
```

避免：

```text
同时转身、撩发、抬手、跨步、低头、拿道具、回头。
```

### 11.4 必须补充光线落点

示例：

```text
柔和自然光落在人物脸侧、肩颈和针织纹理上。
冷白侧光照亮锁骨、腰线和衣料边缘。
侧逆光在发丝、披帛和头饰上形成细腻高光。
玻璃橱窗反射光让脸部和西装廓形更清晰。
```

### 11.5 必须补充场景层次

示例：

```text
背景只保留玻璃窗、浅色桌面和咖啡杯三处生活细节。
古风背景只保留廊柱、宫灯和远处帷幔，并浅景深虚化。
电商背景保持浅色纯背景，不添加复杂道具。
```

## 12. 优化强度分级

根据用户要求选择优化强度。

### 12.1 轻度优化

适合：

```text
用户提示词已经不错，只需要润色；
用户明确说不要大改。
```

处理：

```text
保留原结构；
补少量细节；
修正明显冲突；
补负面约束。
```

---
### 12.2 中度优化

适合：

```text
提示词有参数但比较机械；
需要增强出图稳定性。
```

处理：

```text
保留全部参数；
重新组织为自然画面；
补充主事件、动作链、光线落点；
强化 route 专属风格。
```

---
### 12.3 深度优化

适合：

```text
提示词混乱；
风格冲突；
机械填参严重；
用户说“按我们之前的效果重写”。
```

处理：

```text
提取并锁定原参数；
确定主 route；
重写为完整最终提示词；
补齐所有稳定项；
输出优化说明和最终版。
```

注意：

```text
深度优化也不能改变用户明确参数。
```

## 19. 冲突修复规则

### 19.1 清冷 + 红金

处理：

```text
红色降为辅色点缀；
金色降为刺绣、头饰或腰封高光；
主体保持冷白；
避免喜庆、热烈、暖黄。
```

### 19.2 纯欲 + 不低俗

处理：

```text
保留清纯脸和成年女性自然吸引力；
服装完整；
姿态稳定；
线条整体表达；
禁止局部特写和低俗低机位。
```

### 19.3 电商 + 强风格滤镜

处理：

```text
电商服装还原优先；
滤镜降低强度；
光线保持低色差；
服装颜色准确。
```

### 19.4 新中式 + 古风

处理：

```text
如果用户写茶室、屏风、盘扣、棉麻 → 新中式；
如果用户写披帛、大袖衫、花钿、仙侠 → 古风仙侠；
不能把两者混成符号堆叠。
```

### 19.5 运动 + 纯欲

处理：

```text
如果主目标是运动，保持健康线条和运动场景；
如果主目标是纯欲，保持克制身形表达；
不得低俗健身照，不得身体局部凝视。
```

## 22. 最终原则

提示词优化工具的最终原则是：

```text
不是重新创作，而是在参数保护基础上增强；
不是把提示词写长，而是让画面逻辑成立；
不是堆更多关键词，而是补充场景、动作、光线和镜头关系；
不是替换用户风格，而是让用户原风格更稳定；
不是为了吸睛牺牲安全，而是成年、安全、清晰、有审美；
不是只指出问题，而是必须给出可复制的优化后提示词。
```
