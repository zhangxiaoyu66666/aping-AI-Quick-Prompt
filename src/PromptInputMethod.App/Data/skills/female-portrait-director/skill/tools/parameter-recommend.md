## 1. 工具定位

参数组合推荐工具用于根据用户指定的风格方向、平台用途、审美目标、出图数量和限制条件，推荐一组或多组稳定、吸睛、可直接调用 Skill 的参数组合。

本工具的核心不是简单列几个关键词，而是：

```text
根据主 route 生成完整参数组合；
保证每组参数内部一致；
避免风格冲突；
避免服装、场景、光线、气质互相打架；
每组参数都能直接交给女性人像提示词导演 Skill 生成最终提示词；
兼顾稳定性、审美完成度和平台传播效果。
```

本工具适合处理：

```text
帮我推荐几组比较吸睛的参数；
推荐 4 组适合小红书的参数；
推荐几组纯欲风参数；
推荐几组清冷古风参数；
推荐一些轻性感但不低俗的组合；
推荐几组人物对镜自拍参数；
推荐几组旅行假日写真参数；
推荐 3 组更偏爆款感的组合；
按这个模板帮我设计 5 组不同风格；
给我一组最推荐的参数组合；
```

## 3. 工具职责

本工具负责：

```text
1. 判断用户需要的主风格 route；
2. 根据主风格生成稳定参数组合；
3. 根据用户数量要求输出对应组数；
4. 每组参数必须包含完整字段；
5. 每组之间必须有明显差异；
6. 每组内部必须风格一致；
7. 根据平台用途强化传播感；
8. 根据用户限制排除不适合内容；
9. 对高风险方向进行安全转译；
10. 输出可直接复制给 Skill 使用的参数组合。
```

本工具不负责：

```text
1. 只列关键词，不给完整参数；
2. 推荐互相冲突的场景和服装；
3. 生成无法执行的复杂动作；
4. 忽略用户指定限制；
5. 把电商推荐成普通写真；
6. 把古风推荐成廉价 cosplay；
7. 每组都只是换颜色但没有真实差异；
8. 只给摘要，不给可复制参数。
```

## 5. 推荐总原则

参数推荐必须遵守以下原则：

```text
1. 一组参数只服务一个主风格；
2. 每组只保留一个主场景；
3. 每组只保留一个主服装方向；
4. 每组只保留一个主气质方向；
5. 每组光线、滤镜和配色必须一致；
6. 每组动作必须简单稳定；
7. 每组必须明确成年女性边界；
8. 每组必须避免未成年感；
9. 每组必须能直接进入 route 生成最终提示词；
10. 多组之间要有差异，而不是机械换词。
```

参数组合的核心标准：

```text
好看 > 吸睛 > 复杂
```

也就是说：

```text
不要为了吸睛推荐过度复杂、低俗、冲突或难以生成的参数；
应优先推荐视觉完成度高、平台友好的组合。
```

## 7. 用户目标识别

先判断用户需要哪类推荐。

### 7.1 单风格多组参数

示例：

```text
推荐几组纯欲风参数
推荐几组清冷古风参数
推荐几组都市时尚参数
```

处理方式：

```text
主 route 固定；
每组变化场景、服装、气质、五官、镜头或光线；
不改变主风格。
```

---
### 7.2 多风格参数包

示例：

```text
帮我推荐一些不同风格的人像参数
想要多一点风格变化
```

处理方式：

```text
每组可调用不同 route；
必须标注对应 route；
每组保持内部一致。
```

---
### 7.3 平台导向参数

示例：

```text
适合小红书的参数
适合 X 传播的组合
适合公众号配图的组合
```

处理方式：

```text
平台用途影响构图、主体占比、吸睛度、留白和画面清晰度；
不能覆盖主风格。
```

---
### 7.4 爆款感参数

示例：

```text
推荐几组爆款感参数
更吸睛一点
更容易让人停留
```

处理方式：

```text
强化视觉记忆点；
提高人物主体清晰度；
服装颜色更有辨识度；
场景更明确；
但不得低俗化、幼态化、过度复杂。
```

---
### 7.5 批量出图参数

示例：

```text
给我 5 组，可以直接生成 5 张独立图片
不要合并
不同服装颜色
```

处理方式：

```text
每组必须可独立生成；
组合之间差异明显；
避免同质化；
输出时标注“第 1 组 / 第 2 组”。
```

## 10. 差异化设计规则

多组参数之间必须至少在以下 3 个维度上有差异：

```text
场景；
服装；
配色；
气质；
五官；
镜头；
光线；
平台用途侧重；
动作姿态；
视觉记忆点。
```

禁止：

```text
只换衣服颜色；
只换场景名称；
每组都用同一个五官方向；
每组都是窗边 + 吊带 + 冷白光；
每组都是古风 + 月白 + 披帛；
每组都没有明确视觉差异。
```

推荐差异化方式：

```text
第 1 组：窗边冷白，清冷克制；
第 2 组：海边步道，清爽假日；
第 3 组：街边浅色墙面，都市轻街头；
第 4 组：卧室床边，生活剧照感；
```

## 11. 稳定性规则

推荐参数必须优先稳定。

### 11.1 姿态稳定

优先推荐：

```text
自然站姿；
轻微侧身；
靠墙站姿；
窗边坐姿；
大腿以上构图；
半身近景；
行走中短暂停下；
手自然放在腿侧 / 桌边 / 衣袖 / 衣摆。
```

避免推荐：

```text
复杂跳跃；
大幅舞蹈动作；
双手复杂交叉；
一边转身一边撩发一边走路；
古风飞天大动作；
运动高难度动作；
低俗低机位；
身体局部特写。
```

## 12. 安全规则

所有参数组合必须满足：

```text
人物为 18-26 岁年轻成年女性；
成年气质明确；
不过度欧美化；
不幼态化；
没有未成年感；
不使用萝莉、童颜性感等表达；
```
