## 1. 工具定位

审查友好 / 安全合规改写工具用于将用户提供的女性人像提示词中容易被误读、容易低俗化、容易触发平台限制或容易导致模型生成不稳定的表达，改写为更安全、更清晰、更稳定、更适合正常图片生成的表达。

本工具的核心不是“绕过限制”，而是：

```text
删除不安全表达；
降低低俗误读；
明确成年边界；
保留用户合法审美目标；
将敏感词转译为安全视觉语言；
让提示词更像正常摄影、人像、服装展示、生活剧照或商业图片。
```

本工具必须以 [../core/safety-boundary.md](../core/safety-boundary.md) 为唯一安全边界。不得把明确成年、非露骨的性感、曲线、泳装、内衣风时尚穿搭或适度露肤自动删除；只修复未成年性化、重点部位裸露、透明服装误读、露骨性行为、非自愿情境和平台明确禁止的内容。

本工具尤其适合处理以下场景：

```text
纯欲曲线生活照；
轻性感生活照；
贴身吊带 / 短款上衣 / 短裤；
身形线条表达；
丰腴曲线表达；
电商贴身服装展示；
床边 / 卧室 / 私密空间误读；
运动身形展示；
古风服装过度暴露误读；
平台提示词被拒或生成跑偏。
```

## 3. 工具职责

本工具负责：

```text
1. 识别原提示词中的高风险表达；
2. 判断风险类型；
3. 保留用户合法创作目标；
4. 删除或转译低俗、暗示性、未成年感、情趣化、局部凝视等表达；
5. 明确成年女性边界；
6. 将“性感”等表达转为安全视觉语言；
7. 将“暴露 / 挑逗”等风险方向转为生活照、服装展示、摄影质感、姿态自然；
8. 补充安全负面约束；
9. 输出安全改写后的可复制提示词。
```

本工具不负责：

```text
1. 帮助绕过平台规则；
2. 保留明显不安全、低俗、色情、未成年化表达；
3. 将未成年感与性感、纯欲、贴身服装绑定；
4. 将身体部位作为低俗凝视对象；
5. 强化挑逗姿势、情趣化服装、裸露或暗示性场景；
6. 用隐晦词替代不安全内容继续表达同一不安全意图；
7. 为违规目标做规避性包装。
```

## 5. 改写总原则

安全改写必须遵守以下原则：

```text
1. 不改变用户合法主风格；
2. 不改变用户安全的核心参数；
3. 删除不安全表达；
4. 将高风险词转译为摄影、服装、姿态、光线、构图语言；
5. 明确成年女性；
6. 避免未成年感；
7. 避免低俗、情色、情趣化、暗示性；
8. 避免身体部位特写和低俗凝视；
9. 避免低俗低机位；
10. 最终输出必须仍然能生成好看的图，而不是只剩限制词。
```

核心转译逻辑：

```text
性感 → 成年女性自然吸引力 / 成熟气质 / 克制身体线条
暴露 → 服装完整但版型贴合 / 适度露肤 / 日常穿搭
诱惑 → 有吸引力但克制 / 眼神安静 / 姿态自然
私房 → 电影生活剧照感 / 居家生活照 / 自然室内摄影
胸部强调 → 胸部自然曲线 / 上半身服装版型 / 整体比例
腿部强调 → 腿部自然比例 / 大腿以上构图 / 整体线条
低腰 / 小腹 → 腰线和小腹自然线条 / 服装版型自然呈现
```

第十步：输出改写后提示词
