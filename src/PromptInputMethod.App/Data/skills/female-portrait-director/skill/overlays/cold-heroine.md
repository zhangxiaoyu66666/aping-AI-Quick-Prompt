## 1. Overlay 定位

清冷女主增强是一个气质增强模块，不是独立主 route。

它的作用是：在不改变用户已选择主风格的前提下，增强人物的清冷、克制、疏离、冷白、安静、高级距离感。

本 overlay 不能替代主风格，只能作为主 route 的辅助修饰。

正确理解：

```text
主 route 决定画面类型；
overlay 只调整人物气质、表情、眼神、色温、姿态和少量光线倾向。
```

例如：

```text
清纯生活照 + 清冷女主增强
= 清纯生活照仍然保留，只是人物更安静、冷白、克制。
都市时尚写真 + 清冷女主增强
= 都市街拍仍然保留，只是人物更冷静、高级、有距离感。
古风仙侠美人图 + 清冷女主增强
= 古风仙侠仍然保留，只是色温更冷、甜度更低、人物更疏离。
```

以下情况不应强行调用本 overlay：

```text

用户要求温柔亲近、甜美自然；
用户明确要求笑容灿烂。
```

如果用户同时写：

```text

```

处理方式：

```text

不得把明媚华贵完全改成清冷仙气。
```

## 11. 冲突处理

### 11.1 清冷 + 明媚

如果用户同时写：

```text
清冷、明媚
```

处理：

```text
判断主风格；
如果主风格是明媚华贵，清冷只作为克制表情和降低甜度；
如果主风格是清冷仙气，明媚只作为五官清晰和人物存在感。
```

---
### 11.2 清冷 + 温柔

如果用户同时写：

```text
清冷、温柔
```

处理：

```text
可转译为“清冷但不冷漠，安静温柔，表情克制，眼神柔和但有距离感”。
```

---
### 11.3 清冷 + 纯欲

如果用户同时写：

```text
清冷、纯欲
```

处理：

```text
保留冷白、克制、安静；
身形表达必须更高级、更克制；
不得挑逗、不得私房化、不得身体局部特写。
```

---
### 11.4 清冷 + 运动

如果用户同时写：

```text
清冷、运动
```

处理：

```text
保留运动健康线条和活力场景；
将清冷转译为冷静、自律、干净、低饱和运动感；
不得把运动照变成阴郁冷淡。
```

```text
不是替换主风格，而是在主风格上降低甜度；
不是把所有图都改成清冷古风，而是增强人物的冷白、安静、克制和距离感；
不是病态苍白，也不是假人冷脸，而是高级、沉静、自然的清冷气质；
不是删除用户参数，而是在参数锁定基础上进行气质微调；
不是为了清冷牺牲画面稳定，而是让人物更高级、更克制、更有女主感。
```

所有调用本 overlay 的任务必须遵守本文件。
