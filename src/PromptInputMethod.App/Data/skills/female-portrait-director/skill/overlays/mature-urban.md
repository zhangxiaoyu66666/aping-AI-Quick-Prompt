## 1. Overlay 定位

轻熟都市女性增强是一个气质增强模块，不是独立主 route。

它的作用是：在不改变用户已选择主风格的前提下，增强人物的轻熟感、都市感、优雅感、成熟但不老气的女性气质、稳定姿态、现代生活方式气息和克制女人味。

本 overlay 不能替代主风格，只能作为主 route 的辅助修饰。

正确理解：

```text
主 route 决定画面类型；
overlay 只增强人物的轻熟气质、现代都市感、穿搭质感、姿态稳定性、眼神成熟度和生活方式完成度。
```

例如：

```text
清纯生活照 + 轻熟都市女性增强
= 清纯生活照仍然保留，只是人物更成熟一点、更有轻熟女人味，但不低俗、不私房。
都市时尚写真 + 轻熟都市女性增强
= 都市街拍仍然保留，只是穿搭更优雅、更通勤、更成熟稳定。
法式慵懒写真 + 轻熟都市女性增强
= 法式慵懒仍然保留，只是人物更像成熟都市女性，松弛但不随意。
纯欲曲线生活照 + 轻熟都市女性增强
= 纯欲曲线仍然保留，但人物气质更成熟、更克制、更有都市生活感。
```

## 5. 不适合场景

以下情况不应强行调用本 overlay：

```text
用户明确要求元气少女、甜酷、强运动活力；
用户明确要求清冷疏离、高级冷淡；
用户明确要求明媚华贵古风、盛唐红金；
用户明确要求极简禅意、素雅到极致；
用户要求电商主图但未指定轻熟模特气质；
用户明确要求不要成熟感、不要女人味；
用户明确要求青春校园、学生感、少女感。
```

如果用户同时写：

```text

```

处理方式：

```text
转译为“清纯干净底色 + 成熟稳定气质 + 轻熟女人味”，不得幼态化，也不得低俗化。
```

如果用户同时写：

```text
轻熟都市 + 冷艳御姐
```

处理方式：

```text
判断主目标；
如果轻熟为主，冷艳只增强眼神和穿搭利落感；
如果冷艳御姐为主，轻熟只降低攻击性并增强生活方式感。
```
