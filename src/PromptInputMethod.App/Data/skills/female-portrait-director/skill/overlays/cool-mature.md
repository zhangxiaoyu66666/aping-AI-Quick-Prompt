## 1. Overlay 定位

冷艳御姐增强是一个气质增强模块，不是独立主 route。

它的作用是：在不改变用户已选择主风格的前提下，增强人物的成熟感、冷艳感、控制力、强气场、高级冷淡气质和都市御姐感。

本 overlay 不能替代主风格，只能作为主 route 的辅助修饰。

正确理解：

```text
主 route 决定画面类型；
overlay 只调整人物气质、眼神、表情、姿态、妆容强度、服装气场和少量光线色调。
```

例如：

```text
都市时尚写真 + 冷艳御姐增强
= 都市街拍仍然保留，只是人物更成熟、更冷艳、更有控制力。
复古港风写真 + 冷艳御姐增强
= 港风主结构仍然保留，只是人物更有港片冷艳女主感。
纯欲曲线生活照 + 冷艳御姐增强
= 纯欲曲线仍然保留，但人物气质更成熟冷感，线条表达更克制高级。
影楼精修写真 + 冷艳御姐增强
= 棚拍精修仍然保留，只是人物更冷艳、更有高级肖像气场。
```

## 5. 不适合场景

以下情况不应强行调用本 overlay：

```text
用户明确要求清纯、温柔、初恋感、邻家感；
用户明确要求甜美、元气、阳光、活力；
用户明确要求法式柔和、松弛亲近；
用户明确要求电商主图但未指定冷艳气质；
用户要求自然不抢镜；
用户明确要求少女感、可爱、亲切、柔软。
```

如果用户同时写：

```text
冷艳御姐 + 清纯
```

处理方式：

```text
判断主目标；
如果清纯是主风格，冷艳只能轻微提升成熟度和眼神稳定感；
如果冷艳御姐是主目标，清纯只能作为干净底色，不能变成甜美少女感。
```

如果用户同时写：

```text
冷艳御姐 + 温柔姐姐
```

处理方式：

```text
可转译为“成熟冷静但不强硬，温柔感降低攻击性，整体是克制、有控制力、成熟耐看的女性气质”。
```
