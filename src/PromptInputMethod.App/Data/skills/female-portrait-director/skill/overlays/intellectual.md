## 1. Overlay 定位

高智感知识女性增强是一个气质增强模块，不是独立主 route。

它的作用是：在不改变用户已选择主风格的前提下，增强人物的理性、知性、冷静、书卷气、观察力、专业感、稳定气质和高级思考感。

本 overlay 不能替代主风格，只能作为主 route 的辅助修饰。

正确理解：

```text
主 route 决定画面类型；
overlay 只增强人物的知性气质、眼神状态、姿态端正度、场景细节、服装克制感和整体理性氛围。
```

例如：

```text
清纯生活照 + 高智感知识女性增强
= 清纯生活照仍然保留，只是人物更安静、知性、有书卷气。
都市时尚写真 + 高智感知识女性增强
= 都市街拍仍然保留，只是穿搭更简洁理性，眼神更冷静专业。
新中式东方写真 + 高智感知识女性增强
= 新中式仍然保留，只是人物更有东方书卷气和理性克制感。
古风仙侠美人图 + 高智感知识女性增强
= 古风仙侠仍然保留，只是人物更端正、冷静、有古典书卷气。
```

## 5. 不适合场景

以下情况不应强行调用本 overlay：

```text
用户明确要求甜美、元气、活泼、俏皮；
用户明确要求性感妩媚、纯粹吸睛、强身体吸引力；
用户明确要求明艳华贵、红金盛大、强女主登场；
用户要求电商主图但未指定知性气质；
用户明确要求运动爆发力、强活力、阳光少女感；
用户明确要求不要书卷气、不要职场感、不要理性冷静。
```

如果用户同时写：

```text
高智感 + 纯欲
```

处理方式：

```text
主 route 按用户明确风格判断；
若纯欲为主，高智感只用于眼神、表情和姿态克制；
不得把画面改成书房职业照；
不得削弱用户锁定的场景和服装。
```

如果用户同时写：

```text
高智感 + 明艳
```

处理方式：

```text
可转译为“明艳但有思考感，五官有存在感但表情冷静，眼神聪明稳定，不媚俗”。
```

## 13. 最终原则

高智感知识女性增强 overlay 的最终原则是：

```text
不是替换主风格，而是在主风格上增强理性、知性和书卷气；
不是办公室职业照，也不是学生照，而是成熟、独立、有思考感的知识女性气质；

不是老气和刻板，而是聪明、稳定、耐看、有文化感；
不是删除用户参数，而是在参数锁定基础上进行气质微调。
```
