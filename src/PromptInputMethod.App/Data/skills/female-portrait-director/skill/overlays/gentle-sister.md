## 1. Overlay 定位

温柔姐姐感增强是一个气质增强模块，不是独立主 route。

它的作用是：在不改变用户已选择主风格的前提下，增强人物的成熟温柔、亲近感、稳定感、柔和气质和生活化亲和力。

本 overlay 不能替代主风格，只能作为主 route 的辅助修饰。

正确理解：

```text
主 route 决定画面类型；
overlay 只调整人物气质、表情、眼神、姿态、妆容柔和度、光线温度和亲和感。
```

例如：

```text
清纯生活照 + 温柔姐姐感增强
= 清纯生活照仍然保留，只是人物更成熟温柔、更亲近、更稳定。
法式慵懒写真 + 温柔姐姐感增强
= 法式慵懒仍然保留，只是人物更柔和、更成熟、更有陪伴感。
都市时尚写真 + 温柔姐姐感增强
= 都市街拍仍然保留，只是气场没有那么冷硬，人物更温和亲近。
东方丰腴写真 + 温柔姐姐感增强
= 东方丰腴仍然保留，只是气质更柔润、成熟、亲和。
```

## 5. 不适合场景

以下情况不应强行调用本 overlay：

```text
用户明确要求清冷、疏离、高冷、冷艳；
用户明确要求强气场、御姐、冷淡、锋利；
用户明确要求明艳大女主、强视觉冲击；
用户明确要求运动活力、元气阳光；
用户要求电商主图但未指定温柔模特气质；
用户明确要求少女感、甜酷、叛逆、街头个性。
```

如果用户同时写：

```text
清冷 + 温柔姐姐感
```

处理方式：

```text
可转译为“清冷但不冷漠，温柔但不甜腻，成熟安静，有距离感但不疏远”。
```

如果用户同时写：

```text
冷艳御姐 + 温柔姐姐感
```

处理方式：

```text
判断主目标；
如果冷艳御姐为主，温柔只降低攻击性；
如果温柔姐姐为主，冷艳只增强成熟和控制感。
```

## 13. 最终原则

温柔姐姐感增强 overlay 的最终原则是：

```text
不是替换主风格，而是在主风格上增强成熟温柔；
不是老气，也不是低俗成熟，而是自然、亲近、稳定、耐看的姐姐感；
不是卖萌甜妹，也不是冷硬御姐，而是柔和、成熟、有陪伴感；
不是删除用户参数，而是在参数锁定基础上进行气质微调；
不是为了温柔牺牲风格辨识度，而是让人物更亲近、更自然、更稳定。
```

所有调用本 overlay 的任务必须遵守本文件。
