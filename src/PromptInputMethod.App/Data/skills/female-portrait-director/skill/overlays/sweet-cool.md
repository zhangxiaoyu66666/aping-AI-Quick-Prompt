## 1. Overlay 定位

甜酷年轻女性增强是一个气质增强模块，不是独立主 route。

它的作用是：在不改变用户已选择主风格的前提下，增强人物的年轻感、个性、甜酷反差、街头感、轻时尚感、活力和社交平台吸睛度。

本 overlay 不能替代主风格，只能作为主 route 的辅助修饰。

正确理解：

```text
主 route 决定画面类型；
overlay 只增强人物气质、眼神、姿态、穿搭细节、色彩活力和年轻个性感。
```

例如：

```text
都市时尚写真 + 甜酷年轻女性增强
= 都市街拍仍然保留，只是人物更年轻、更有街头个性和甜酷感。
活力运动写真 + 甜酷年轻女性增强
= 运动写真仍然保留，只是人物更有年轻时尚感和社交平台传播感。
清纯生活照 + 甜酷年轻女性增强
= 清纯生活照仍然保留，只是人物更俏皮一点、更有个性，但不能幼态化。
旅行假日写真 + 甜酷年轻女性增强
= 旅行假日仍然保留，只是穿搭更年轻、有活力、有记忆点。
```

## 5. 不适合场景

以下情况不应强行调用本 overlay：

```text
用户明确要求清冷、疏离、高级冷淡；
用户明确要求成熟温柔、姐姐感、知性；
用户明确要求古典端庄、东方留白、禅意极简；
用户明确要求明媚华贵古风、清冷仙气古风；
用户要求电商主图但未指定甜酷模特气质；
用户明确要求高级商务、强气场御姐、知识女性；
用户要求不要年轻感、不要俏皮感。
```

如果用户同时写：

```text
甜酷 + 清冷
```

处理方式：

```text
可转译为“冷白清爽的甜酷感，眼神安静但穿搭更有个性”，不得变成完全冷艳或幼态甜妹。
```

如果用户同时写：

```text
甜酷 + 温柔姐姐
```

处理方式：

```text
判断主目标；
如果温柔姐姐为主，甜酷只作为穿搭小细节；
如果甜酷为主，温柔只降低攻击性和街头硬度。
```

## 11. 冲突处理

### 11.1 甜酷 + 清冷

如果用户同时写：

```text
甜酷、清冷
```

处理：

```text
转译为“冷白清爽的甜酷感，眼神安静但穿搭更有个性”，保留成年感，不卖萌。
```

---
### 11.2 甜酷 + 温柔姐姐

如果用户同时写：

```text
甜酷、温柔姐姐
```

处理：

```text
判断主目标；
如果温柔姐姐为主，甜酷只作为轻微穿搭个性；
如果甜酷为主，温柔只降低攻击性，让人物更亲和。
```

---
### 11.3 甜酷 + 纯欲

如果用户同时写：

```text
甜酷、纯欲
```

处理：

```text
保留纯欲 route 的成年边界和克制线条；
甜酷只增强年轻个性、色彩和穿搭态度；
不得幼态化、不得低俗化。
```

---
### 11.4 甜酷 + 运动

如果用户同时写：

```text
甜酷、运动
```

处理：

```text
两者兼容度高；
保留运动健康线条和场景；
增强年轻潮流感、运动鞋、棒球帽、小面积亮色和自信表情。
```

---
### 11.5 甜酷 + 电商

如果用户同时写：

```text
甜酷、电商主图
```

处理：

```text
主 route 必须仍为 ecommerce-tryon；
甜酷只作为模特气质；
不得让配饰、姿势或背景遮挡服装；
服装还原第一优先。
```

## 13. 最终原则

甜酷年轻女性增强 overlay 的最终原则是：

```text
不是替换主风格，而是在主风格上增强年轻个性和甜酷态度；
不是幼态少女感，也不是学生感，而是明确成年女性的清爽、活力、个性；
不是低俗甜酷，而是穿搭、眼神、姿态和色彩带来的年轻吸睛感；
不是潮流元素乱堆，而是少量配饰和清爽色彩提升记忆点；
不是删除用户参数，而是在参数锁定基础上进行气质微调；
不是为了吸睛牺牲画面稳定，而是让人物更年轻、更鲜活、更适合社交平台传播。
```
