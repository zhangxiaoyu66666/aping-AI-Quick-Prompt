## 1. Overlay 定位

明艳女主增强是一个气质增强模块，不是独立主 route。

它的作用是：在不改变用户已选择主风格的前提下，增强人物的明艳度、五官存在感、眼神张力、女主气场、妆造完成度和视觉记忆点。

本 overlay 不能替代主风格，只能作为主 route 的辅助修饰。

正确理解：

```text
主 route 决定画面类型；
overlay 只增强人物的明艳度、眼神、妆容、表情、姿态、主体占比和局部光线高光。
```

例如：

```text
清纯生活照 + 明艳女主增强
= 清纯生活照仍然保留，只是人物更有记忆点、眉眼更清晰、气质更像女主。
都市时尚写真 + 明艳女主增强
= 都市街拍仍然保留，只是人物更有气场、更抓眼。
古风仙侠美人图 + 明艳女主增强
= 古风仙侠仍然保留，只是妆造更精致、眼神更有女主感。
复古港风写真 + 明艳女主增强
= 港风主结构仍然保留，只是更像旧港片女主。
```

## 5. 不适合场景

以下情况不应强行调用本 overlay：

```text
用户明确要求清冷、疏离、低调、素雅、寡淡；
用户明确要求自然淡颜、生活感、不抢镜；
用户明确要求电商服装主图且未指定明艳气质；
用户明确要求极简、禅意、留白、低存在感；
用户明确要求运动健康自然，不要妆感太强；
用户明确要求不要浓妆、不要女主感。
```

如果用户同时写：

```text
清冷 + 明艳
```

处理方式：

```text
判断主风格；
如果清冷是主目标，明艳只增强五官清晰度和人物存在感；
如果明艳是主目标，清冷只降低甜腻感和艳俗感。
```

## 13. 最终原则

明艳女主增强 overlay 的最终原则是：

```text
不是替换主风格，而是在主风格上提高人物存在感；
不是把所有图都改成明媚华贵古风，而是增强五官、眼神、妆容和女主气场；
不是低俗性感，也不是网红浓妆，而是高级、自然、有辨识度的明艳；
不是删除用户参数，而是在参数锁定基础上进行气质增强；
不是为了吸睛牺牲画面稳定，而是让人物更像视觉中心、更有记忆点。
```

所有调用本 overlay 的任务必须遵守本文件。
