# 公开调用说明

这个 Skill 会把少量女性人像参数扩写成完整、稳定、可直接复制的 AI 生图提示词，也可以在保留授权自拍五官或产品核心视觉的基础上直接生成目标风格图片。它会先完成内部导演设计，再锁定并扩写年龄特征、五官、服装、身体线条、姿态、场景、镜头、光线和滤镜，最后把这些模块融合为同一个自然瞬间。

系统通过 [style-registry.md](style-registry.md) 匹配唯一主 Route，并且只按需读取命中的风格文件。每次生成前必须先通过 [core/director-gate.md](core/director-gate.md) 完成内部导演设计。输出不是参数摘要，也不是字段填空，而是经过导演式思考后的完整画面扩写。

## 输出模式

- `简洁`：只返回最终提示词和负面约束。
- `标准`：默认模式。返回参数锁定、逐段导演式模块扩写、五段式最终提示词和负面约束。模块扩写至少包含年龄与五官、身形与仪态、服装配色与发饰、华丽度的可见设计、场景、镜头、光线与滤镜。最终提示词和负面约束分别使用独立 Markdown `text` 代码块，便于直接复制。
- `导演详版`：在标准模式基础上，进一步说明瞬间、动作链、选择性环境细节、参数扩写关系、镜头目的与光线设计。
- `直接生成图片`：内部扩写后直接调用可用的图片生成能力。
- `参考图直接生成`：上传自拍或产品图，锁定主体后直接生成目标风格图片，默认不输出内部提示词。

直接说 `只要最终提示词` 等同于选择 `简洁`。

最终提示词第一句必须写明画幅比例。用户额外填写像素尺寸、分辨率或宽高时，系统必须保留原值并一并写入第一句；用户未填写时不得虚构具体像素值。

## 参考图直接生成

```text
保留我的五官，生成复古港风写真
用我的自拍做一张古风仙侠图
保持这个产品不变，生成一张高级广告图
保留我的脸，穿上第二张图里的衣服
```

人物参考图仅支持本人或已授权的成年人物。产品默认锁定品类、轮廓、颜色、材质、图案和关键结构。Logo 与小字尽量保留，但不承诺逐字准确。

## 使用边界

文本生图默认人物必须是虚构、明确成年的女性。参考图编辑允许本人或已授权成年人物。完整边界见 [../docs/prompt_safety.md](../docs/prompt_safety.md)。
